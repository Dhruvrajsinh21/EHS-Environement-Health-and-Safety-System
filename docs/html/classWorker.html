Detailed Description

Represents a worker in the EHS management system.

Inherits from the User class. A worker can:

  • Report task completion with media and description.
  • Provide feedback on rules.

Allows a worker to report task work.

Worker reports progress on a task.

Lists assigned tasks, accepts a report description and media path, saves the media, and updates the task in the database.

Parameters

This method allows a worker to report their progress on a task. It uses threading to handle multiple task reports concurrently, allowing workers to submit reports simultaneously without waiting for each other.

Parameters

OOP Principles: The Worker class inherits from User and uses member functions to encapsulate behavior like reporting tasks and providing feedback.

Threading: Threads are used to handle task reporting concurrently so multiple workers can submit reports at the same time.

Encapsulation: Task reporting is encapsulated within the reportTaskWork function. The database interactions are isolated inside the method, making it easy to manage the worker's task reporting logic separately.
