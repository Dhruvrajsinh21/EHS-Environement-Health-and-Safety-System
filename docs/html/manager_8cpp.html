Detailed Description

Implements the Manager class functionalities for the EHS Management System.

This file defines the behavior of the Manager role, including:

  • Assigning tasks to workers.
  • Reporting violations related to tasks.
  • Adding and deleting safety rules.
  • Deleting existing tasks.

The class uses SQLite for all database operations and follows object-oriented principles. It is intended to be used in conjunction with the overall EHS system that manages Environment, Health, and Safety processes.
