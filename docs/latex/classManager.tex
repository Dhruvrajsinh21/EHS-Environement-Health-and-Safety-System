\doxysubsection{Detailed Description}
Represents a manager in the EHS management system. 

Inherits from the \mbox{\hyperlink{classUser}{User}} class. A manager can\+:
\begin{DoxyItemize}
\item Assign tasks to workers
\item Report safety violations
\item Add or delete safety rules
\item Delete tasks 
\end{DoxyItemize}
