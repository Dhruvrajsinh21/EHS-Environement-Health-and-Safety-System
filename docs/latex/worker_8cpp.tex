\doxysubsection{Detailed Description}
Implementation of the \mbox{\hyperlink{classWorker}{Worker}} class in the EHS Management System. 

This file defines the behavior of the \mbox{\hyperlink{classWorker}{Worker}} class, including task reporting and feedback submission functionalities. Workers can report task progress, submit media files, and provide feedback on safety rules. Multithreading is used for concurrent task reporting, and database access is synchronized using mutexes.

\begin{DoxyItemize}
\item Inheritance\+: \mbox{\hyperlink{classWorker}{Worker}} inherits from \mbox{\hyperlink{classUser}{User}}.
\item Encapsulation\+: \mbox{\hyperlink{classWorker}{Worker}} functionalities are organized in class methods.
\end{DoxyItemize}

Multithreading\+:
\begin{DoxyItemize}
\item Task reporting is performed using threads to simulate concurrent workers.
\item Mutex is used to synchronize database operations. 
\end{DoxyItemize}
