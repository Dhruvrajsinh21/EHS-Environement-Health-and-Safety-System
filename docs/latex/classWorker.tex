\doxysubsection{Detailed Description}
Represents a worker in the EHS management system. 

Inherits from the \mbox{\hyperlink{classUser}{User}} class. A worker can\+:
\begin{DoxyItemize}
\item Report task completion with media and description.
\item Provide feedback on rules. 
\end{DoxyItemize}
